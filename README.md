# 夜局（本地单机）

当前稳定版本：`v1.0.0`

一个完整本地运行的本地单机扑克项目：
- 1 名人类玩家 vs 1~10 名 AI
- 标准德州（52 张）
- 短牌德州 6+（36 张，移除 2~5）
- 奥马哈（4 张底牌，摊牌必须使用 2 张底牌 + 3 张公共牌）
- PLO（底池限注奥马哈，下注/加注上限受底池约束）
- 梭哈（五张梭哈，无公共牌，按街发牌）
- 斗地主（独立本地模式，1 人类 + 2 AI，叫分抢地主、炸弹/王炸倍率、春天结算）
- 掼蛋（独立本地模式，1 人类 + 3 AI，对家配合、级牌推进、基础炸弹/钢板/连对）
- 现金局模式（固定盲注）
- 锦标赛模式（每 N 手自动升盲）
- 锦标赛盲注结构预设（标准 / Turbo / Deep）
- 锦标赛自动前注（Ante，按盲注结构档位配置）
- 锦标赛奖励结构（按参赛人数自动生成奖励圈与名次占比）
- 现金局可选 UTG 跨注（Straddle，默认 2x 大盲）
- 快速模式（手牌结束后自动开始下一手）
- 本地自动存档与继续对局（返回菜单或刷新页面后可恢复当前会话）
- 本地生涯战绩档案（跨会话持久化）
- 生涯战绩按玩法拆分（标准 / 短牌 / 奥马哈 / PLO / 梭哈）
- 生涯战绩支持导出 JSON、导入恢复与一键清空
- AI 难度档位（保守 / 标准 / 激进）
- 局中可热切换 AI 难度（HUD）
- 本地音效系统（发牌 / 入池 / 公共牌揭示 / 轮到你 / 摊牌 / 全下 / 淘汰 / 结算），支持 `关 / 柔 / 满` 三档切换
- 自动行动预设（过/弃到我、仅过牌、跟/过到我、跟注阈值）
- 自动行动增强（跟/过≤XBB，否则弃牌）
- 局中锦标赛详情抽屉（完整盲注表 + 奖励结构）
- 锦标赛压力分析（M 值 / BB 深度 / 泡沫位 / 下一级压力提示）
- 历史中心结果中心（冠军数 / 进圈次数 / 最佳名次 / 最近完成会话）
- 历史中心结果中心支持按玩法拆分统计、生涯档案导入/导出/清空，以及局制/最近难度分布概览
- 本地回放归档（最近 8 个已结束/被替换会话，支持跨会话逐手回放）
- 本地回放归档已迁移为 `IndexedDB` 主存储 + `localStorage` bootstrap 回退，长历史加载更稳
- 回放归档支持导出 JSON、合并导入、覆盖导入与一键清空
- 回放归档导入前提供差异预检面板（当前/导入后会话数、重复手牌数、新增/重叠/移除会话）
- 回放归档导入支持按会话选择子集，只导入本次勾选的会话
- 回放归档预检面板提供“会话差异表”，按会话显示当前手数、导入手数、新增手数、重复手数与导入后状态
- 回放归档差异表继续细化到 `handId` 明细，可直接看到新增手号、重复手号和导入后保留的手号
- 回放归档差异表中的 `handId` 可直接点击预览该手回放，返回后保留原导入预检状态
- 返回导入预检后会自动滚回刚查看过的 `handId`，并高亮对应手号与差异行
- 导入预检中新增“最近查看手牌”列表，可在最近几手之间快速来回切换回放
- 导入预检支持“复盘篮子”，可把重点手牌固定保留，便于在多手之间反复横向对比
- 会话详情支持“关键节点跳转”，可从单手卡片直接跳到翻牌 / 全下 / 边池 / 摊牌等重点步骤
- 会话详情支持“关键节点 hover 预览 + mini board 缩略牌面”，无需进入回放即可先看到该节点的大致桌面状态
- 会话详情支持“对比组”，可将多手固定并排，对照盈亏、标签与关键节点，并支持“仅高亮差异”
- 对比组支持“差异摘要 + 排序切换”，可直接看出几手在盈亏 / 底池 / 结果 / 标签 / 关键节点上的主要不同
- 专注模式下注框支持左 / 右停靠切换，不再覆盖手牌与公共牌
- 小屏专注模式会自动切换为“牌桌 / Dock / 面板”纵向堆叠，避免下注框重新压住牌桌
- 牌桌新增动态特效层：公共牌揭示爆发、筹码入池脉冲、胜者扫光、环境漂浮光斑
- 行动玩家座位新增聚光边框与状态脉冲，卡牌新增镜面扫光材质
- 玩家座位新增本地人物形象系统：头像徽记、称号、个性配色，主牌桌与回放统一复用
- 结算层新增人物聚焦卡，胜者会以大头像卡片展示称号、牌型说明和派奖结果
- 锦标赛终局新增冠军人物 banner，带冠军聚光、奖杯徽记和人物主卡；冠军与玩家成绩会在终局总结中单独突出
- 冠军 banner 内含赛制摘要 chip（奖励、参赛人数、总手数、最终盲注），并按序分段入场
- 回放侧栏新增“当前人物聚焦卡”，会随当前步骤切换焦点人物并展示对应状态
- 回放人物聚焦卡加入固定高度壳层和切牌式过渡，切换关键节点时不再生硬跳变，也不会把右侧信息区顶高顶低
- 回放人物卡额外带扫光遮罩层，关键节点切换时更接近“切牌”观感
- 牌桌实时态势面板（筹码排名 / 淘汰记录 / M 值 / 升盲倒计时）
- 态势面板展示奖励圈与后续盲注档位
- 态势榜支持“仅在局/全部”与“按筹码/近5手净变化”切换
- 近N手窗口可切换（3/5/10）并参与态势榜排序
- 严格下注顺序、最小加注、全下与多边池结算
- 事件溯源 + 可视化回放（逐步、自动播放、跳阶段）

## 默认短牌规则说明（可配置）
短牌规则位于：
- `src/engine/evaluators/shortDeckRules.ts`

当前默认预设（`defaultShortDeckPreset`）：
- 使用 36 张牌（2~5 移除）
- `A-6-7-8-9` 计作顺子
- `同花 > 葫芦`
- `三条 > 顺子`

说明：短牌不同赛事存在差异。该项目将短牌牌型顺序独立为规则模块，可替换 `categoryOrder` 快速调整。

## iPad 运行
- 这版已经补齐 `manifest + apple-touch-icon + service worker`，可作为 iPad Web App 安装到主屏幕。
- 推荐方式：
  1. 在本机执行 `npm run build && npm run preview -- --host 0.0.0.0 --port 4183`
  2. 用 iPad Safari 打开你的局域网地址，例如 `http://你的电脑IP:4183`
  3. 点 Safari 的“分享” -> “添加到主屏幕”
  4. 从主屏幕启动，应用会以 `standalone` 方式全屏运行
- 当前已补齐：
  - `viewport-fit=cover`
  - iPad 安装提示条
  - 竖屏时的横屏游玩提示
  - 安全区 `safe-area` padding
  - `service worker` 离线缓存外壳资源
- 说明：
  - iPad 上最佳体验是 `横屏`
  - 如果后续要上架 App Store，再考虑基于当前前端壳接 `Capacitor` 或原生 WebView 容器

## 主要架构
- `src/engine/cards.ts`：牌堆与发牌
- `src/engine/evaluators/standardEvaluator.ts`：标准德州评估器
- `src/engine/evaluators/shortDeckEvaluator.ts`：短牌评估器
- `src/engine/evaluators/omahaEvaluator.ts`：奥马哈/PLO 评估器（2手牌+3公共牌）
- `src/engine/evaluators/shortDeckRules.ts`：短牌规则预设
- `src/engine/actionValidation.ts`：合法动作验证
- `src/engine/bettingRound.ts`：下注轮状态机（顺序、最小加注、重开加注）
- `src/engine/potSettlement.ts`：主池/边池/分池结算
- `src/engine/ai.ts`：AI 决策
- `src/engine/handEngine.ts`：整手流程（翻前->翻牌->转牌->河牌->摊牌/结算）
- `src/replay/replayBuilder.ts`：回放事件建模
- `src/replay/replayRecordKey.ts`：跨会话唯一手牌回放键
- `src/replay/replayReconstructor.ts`：按事件重建桌面快照
- `src/replay/replayAnalysis.ts`：回放分析（关键节点、可疑诈唬线、时间线筛选）
- `src/replay/sessionHistory.ts`：按会话聚合 hand history
- `src/state/replayArchive.ts`：本地回放归档存储（IndexedDB + bootstrap 回退）与去重/截断策略
- `src/state/audioPreferences.ts`：本地音效偏好
- `src/audio/soundBus.ts`：Web Audio 合成音效总线
- `src/ui/components/*`：菜单、牌桌、历史中心、回放查看器

## 功能覆盖
- 完整持续牌局循环，庄位/盲注轮转
- 淘汰机制与比赛结束判定
- 支持平分底池与奇数筹码分配
- 记录每手完整结构化历史：行动序列、边池事件、摊牌结果、派奖明细
- 自动将当前会话保存到浏览器本地存储，可从主菜单“继续上次对局”恢复
- 已结束或被替换的会话手牌会自动沉淀到本地回放归档，并在历史中心的“会话视图”中长期可见
- 回放归档首次渲染走 `localStorage` bootstrap，后台再异步水合 `IndexedDB` 全量数据，避免大归档拖慢首屏
- 回放归档可导出为 JSON 备份，也可通过“合并导入 / 覆盖导入”迁移到当前设备
- 回放归档导入前会显示差异预检：当前归档、导入文件、导入后结果、重复手牌数，以及新增/重叠/移除/溢出 `sessionId`
- 回放归档导入时可在预检面板内勾选具体会话，实时重算导入后结果，再决定是否确认
- 预检中的“会话差异表”会按当前勾选结果逐会话展示 `当前 / 导入 / 新增 / 重复 / 结果 / 状态`，方便确认会新增多少手
- 差异表下方会直接列出每个会话的 `新增 handId / 重复 handId / 导入后保留 handId`
- 点击这些 `handId` 会直接打开该手的回放预览，返回历史中心后仍保持原来的“会话视图 / 导入预检 / 勾选状态”
- 回到预检面板时，刚查看过的 `handId` 会带高亮，并自动滚回可见位置，避免丢定位
- “最近查看手牌”列表会保留最近查看的多手，并支持直接再次打开对应回放
- 导入预检支持“复盘篮子”，可将重点 `handId` 固定保留，并从篮子中直接重开回放继续比对
- 会话详情会为每手生成“关键节点”按钮，点击后直接以目标步骤打开回放，而不是只能从第 0 步开始拖动
- 关键节点按钮支持 hover / focus 预览，会显示该节点的街道、底池、公共牌数量、边池数量、mini board 缩略图与桌面状态说明
- 会话详情支持“加入对比”，可把多手放入对比组并排展示，再通过“仅高亮差异”快速聚焦不同的盲注 / 盈亏 / 标签 / 关键节点
- 对比组顶部会生成“差异摘要”，自动汇总几手在 `盈亏 / 底池 / 结果 / 标签 / 教学 / 关键节点` 上的不同，并支持 `篮子顺序 / 最新优先 / 盈亏波动 / 大底池优先` 排序
- 专注模式点击“操作”会打开侧边下注框，并支持左 / 右停靠切换，牌桌中央区域保持完整可视
- 窄屏下专注模式面板不再绝对定位覆盖牌桌，而是自动下沉到牌桌下方，保留完整桌面可视区域
- 牌桌中央新增 `TableEffectsLayer`，按 `board / pot / settlement` 变化触发阶段性特效，不依赖后端和外部素材
- `SeatPanel` 集成 `PlayerPortrait`，为人类玩家与 AI 提供稳定人物画像，不依赖外部图片资源
- 人物画像支持 `专注 / 全下 / 弃牌 / 获胜 / 淘汰` 情绪态，主牌桌与回放共用同一套视觉反馈
- 情绪态已接入额外镜面 flare 演出，并和 `摊牌 / 全下 / 淘汰 / 结算` 的本地音效挂点联动
- 当前项目未内置本地采样音资源，因此音效仍以 Web Audio 合成音实现；后续如补采样资源，可直接沿用现有 cue 名称接入
- 主菜单支持切换人类玩家主形象主题，当前提供 `牌桌主理人 / 彗光领航 / 轨迹策士 / 夜幕策展`
- 头像主题会写入浏览器本地偏好，并随新开对局、继续对局和历史回放保持一致
- 人物画像布局已针对 `1~10` AI 做碰撞回归，确认头像不会压住公共牌、主底池或玩家手牌
- 锦标赛终局冠军 banner 现包含 `冠军奖励 / 参赛人数 / 总手数 / 最终盲注` 四个摘要 chip，并带有从牌桌中心卷入式的尾焰入场
- 回放右侧人物聚焦卡的切牌式 wipe 已按 `翻前 / 翻牌 / 转牌 / 河牌 / 摊牌 / 结算` 切换色调，并叠加轻量环境光，不改变卡片高度
- HUD 新增 `动效：满 / 柔 / 减` 循环切换，偏好写入浏览器本地存储；若系统启用 `prefers-reduced-motion`，首次进入会自动采用减弱动效
- 动效强度会影响牌桌环境粒子、胜者扫光、冠军摘要粒子、回放人物卡 wipe 与环境光等装饰层，但不会改变核心信息布局
- 回放右栏标题、边框和当前激活事件也会按当前街道做低强度色调联动，保持信息可读同时强化阶段感
- 现已支持 4 套全局牌桌皮肤：`霓虹深海 / 黑金高厅 / 祖母绿毡 / 绯红王座`
- 主题皮肤会同步影响主菜单、HUD、牌桌桌毡、公共牌托盘、底池、座位面板等主视觉区域，并可在主菜单或局内 HUD 一键切换
- 牌桌主题偏好写入浏览器本地存储，并随继续对局、历史中心和回放界面保持一致
- HUD 音效按钮会循环切换 `柔 -> 满 -> 关`，偏好写入浏览器本地存储
- 回放中可逐步查看每一步桌面状态与时间线
