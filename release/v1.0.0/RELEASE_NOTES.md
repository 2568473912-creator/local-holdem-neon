# Release 1.0.0

夜局 `v1.0.0` 首个稳定发布版本。

## 主要功能
- 标准德州 + 短牌德州（独立评估器）
- 现金局 + 锦标赛（升盲、前注）
- 完整下注规则与边池结算
- AI 风格与难度档位（可局中切换）
- 历史中心 + 可视化回放
- 回放关键节点筛选、高压阈值、组合时间线筛选
- 牌桌态势榜（仅在局/全部、近N手净变化）

## 发布资产
- `yeju-v1.0.0-windows.zip`
  - `dist/` 静态文件
  - `启动游戏.bat` / `启动游戏.ps1`
  - Windows 运行说明文档
- `yeju-v1.0.0-windows-portable.zip`
  - `Yeju.exe`（免 Node 一键运行）
  - 内置打包资源（Electron + dist）
  - 便携版使用说明与用户说明书

## 兼容性修复（Windows 便携版）
- 修复 Electron 便携版在部分 Windows 环境打开后白屏的问题。
