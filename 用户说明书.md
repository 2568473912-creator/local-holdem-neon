# 霓虹德州俱乐部 用户说明书（Release 1.0）

## 1. 产品概览
`霓虹德州俱乐部` 是一个纯前端、可离线本地运行的单机德州扑克牌桌项目。

- 支持 `标准德州`（52 张牌）
- 支持 `短牌德州`（6+，36 张牌）
- 1 名人类玩家对战 1-10 名 AI
- 支持现金局与锦标赛（升盲与前注）
- 支持完整回放与历史复盘

## 2. 运行环境

### 2.1 开发/源码运行
- Node.js 18+（建议 20+）
- npm 9+

### 2.2 Windows Release 运行
Windows 打包版优先支持以下任一环境：
- Python 3（推荐，内置 http.server）
- 或 Node.js（通过 npx 启动静态服务）

## 3. 本地启动（源码）
在项目根目录执行：

```bash
npm install
npm run dev
```

默认在浏览器打开 `http://127.0.0.1:5173`（端口以终端输出为准）。

## 4. 游戏流程说明

### 4.1 主菜单配置
- 游戏模式：标准德州 / 短牌德州
- 对局模式：现金局 / 锦标赛
- AI 难度：保守 / 标准 / 激进
- AI 人数：1-10
- 初始筹码
- 盲注档位
- 锦标赛升盲间隔
- 节奏模式（标准/快速）

### 4.2 牌桌操作
- `弃牌 / 过牌 / 跟注 / 加注 / 全下`
- 加注滑条与快捷尺寸（1/2 池、2/3 池、底池、2x 底池）
- 快捷键：
  - `F` 弃牌
  - `Q` 过牌
  - `C` 跟注
  - `A` 全下
  - `R` 下注/加注
  - `1/2/3/4` 下注档位

### 4.3 HUD 与对局控制
- 暂停/继续
- 下一手
- 历史回放
- 重新开局
- 返回菜单
- 局中 AI 难度热切换

## 5. 规则与结算
程序已实现完整下注流程与严格校验：

- 正确行动顺序与轮转
- 最小加注规则
- 短码全下不重开加注场景
- 主池/边池/多边池结算
- 平分池与奇数筹码分配
- 淘汰与比赛结束判定
- 标准德州与短牌德州评估器独立维护

## 6. 回放与复盘
每手牌会记录完整事件数据，并支持可视化回放：

- 逐步前进/后退
- 自动播放
- 跳转街道（翻前/翻牌/转牌/河牌/摊牌）
- 关键节点筛选（全部/高压/诈唬线/淘汰/结算）
- 高压阈值（2/4/6/8/10BB）
- 时间线组合筛选（高压/诈唬/淘汰/教学/摊牌）
- 筹码变化（起始 -> 结束）

## 7. 牌桌态势面板
- 在局人数、平均筹码、你的排名、M 值
- 盲注节奏提示
- 筹码榜显示：
  - 仅在局 / 全部
  - 按筹码 / 近N手净变化
  - N 窗口：3/5/10 手

## 8. Windows Release 1.0 使用
发布包目录中提供：

- `dist/` 前端静态资源
- `启动游戏.bat`（推荐）
- `启动游戏.ps1`

双击 `启动游戏.bat` 后：
1. 自动尝试使用 Python 启动本地服务（优先）
2. 若无 Python，再尝试 Node+npx
3. 自动打开浏览器 `http://127.0.0.1:4175`

关闭服务：在启动脚本的终端窗口按 `Ctrl + C`。

## 9. 常见问题

### 9.1 页面打不开
- 检查端口 `4175` 是否被占用
- 关闭安全软件对本地端口拦截
- 手动访问 `http://127.0.0.1:4175`

### 9.2 启动脚本报错找不到 Python/Node
- 安装 Python 3 或 Node.js 18+
- 安装后重开终端/资源管理器再运行脚本

### 9.3 操作按钮灰掉
- 当前不是你的行动回合
- 或该动作在规则上非法（最小加注/再加注权限等）
