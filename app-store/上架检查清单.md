# iPad App Store 上架检查清单

## 二进制与工程
- [ ] 安装完整 Xcode.app
- [ ] `xcode-select` 指向 `/Applications/Xcode.app/Contents/Developer`
- [ ] `npm run build`
- [ ] `npm run mobile:ios:sync`
- [ ] 在 Xcode 中打开 `ios/App/App.xcodeproj`
- [ ] 设置真实可用的 Bundle ID
- [ ] 选择有效 Team 并完成签名
- [ ] 在真机 iPad 上跑一遍

## App Store Connect
- [ ] 新建 App 记录
- [ ] 填写名称、副标题、描述、关键词
- [ ] 设置主分类/副分类
- [ ] 完成年龄分级问卷
- [ ] App Privacy 如实填写
- [ ] 填写 Support URL
- [ ] 填写 Privacy Policy URL
- [ ] 补真实开发者联系信息

## 隐私与合规
- [ ] 确认当前版本仍未接入广告/统计/推送/账号 SDK
- [ ] 如果确实未收集数据，在 App Privacy 中选择 “No, we do not collect data from this app”
- [ ] 如果接入了任何联网分析或云存档，先更新隐私政策与 App Privacy
- [ ] 元数据中避免使用“真钱”“提现”“博彩平台”等表述

## 素材
- [ ] 运行 `npm run appstore:shots:ipad`
- [ ] 从 `output/app-store-ipad-13/` 选择 3-10 张截图
- [ ] 检查截图是否完整、无调试提示、无错误弹层
- [ ] 准备最终 App Icon / 启动图

## 审核备注
- [ ] 使用 `app-store/AppReviewNotes.md` 作为 App Review Information 基础模板
- [ ] 明确说明本产品为本地单机娱乐/训练，不涉及真实货币赌博
